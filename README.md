# Run Deno
(fs module used for functional demo, no database reqired)

### `deno task start`
Run Deno REST API


# Run React Dev Mode

### `npm run start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
